# Spam Email Classification using Machine Learning
<br>
This project is a simple NLP-based machine learning model that classifies messages as Spam or Ham (Not Spam) using text data.

Project Overview
-Built a text classification model using email/SMS messages
-Converted text into numerical features using TF-IDF Vectorization
-Trained a Naive Bayes classifier for prediction
-Achieved high accuracy in detecting spam messages
