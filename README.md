# Spam Email Classification using Machine Learning

##  Project Overview
This project is a simple Natural Language Processing (NLP) based machine learning model that classifies messages as **Spam** or **Ham (Not Spam)** using text data.

---

## What this project does
- Builds a text classification model using email/SMS messages  
- Converts text into numerical features using **TF-IDF Vectorization**  
- Trains a **Naive Bayes classifier** for prediction  
- Achieves high accuracy in detecting spam messages  

---


## ⚙️ Workflow
1. Data Collection (SMS/Email dataset)  
2. Data Cleaning & Preprocessing  
3. Feature Extraction using TF-IDF  
4. Model Training (Naive Bayes)  
5. Model Evaluation  
6. Prediction  

---

## 📊 Results
- High accuracy achieved on test data  
- Effective classification of spam vs ham messages
